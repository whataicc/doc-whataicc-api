# Raycast 插件 ChatGPT 使用指南（推荐使用）
---

安装插件, 在 Raycast Store 中找到 ChatGPT 插件，并按照提示安装：



配置 API Key, 安装完成后，在该插件配置中的 API Key 中填入我们的 API Key。选中 Change API Endpoint，并在 API Endpoint 中填入 https://api.whatai.cc/v1。
